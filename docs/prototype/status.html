Order Status Description:


Pending Payment：The user has placed an order but has not made the payment.


Pending Shipment: Payment has been made( Payment callback - Return successful), but the book has not yet been delivered.


Borrowing: The lender has shipped the books and they are currently on loan.


Overdue: The borrowing has exceeded the agreed time.


Deposit Deducted: The deposit will be deducted due to overdue or damaged condition.


Returned: The borrower has returned the books and is awaiting confirmation.


Completed: The entire process is complete (confirmation of return + fund settlement)

Book Status Description:


Listed: = already on shelf/available.


Lent Out: =“On Loan”. Only in this status, we can find Orders


Unlisted: shows book is not available.

Status Change:


Pending Payment →


Pending Shipment →


Borrowing →


    if (has exceeded the agreed time) → Overdue → Deposit Deducted → Completed


     if (returned on time) → Returned → Completed
