## Visual Design Specifications

### Color Scheme
- **Primary:** #1E3A8A (Professional Blue)
- **Secondary:** #F59E0B (Accent Gold) 
- **Success:** #10B981 (Green for completed items)
- **Warning:** #F59E0B (Orange for pending items)
- **Text:** #1F2937 (Dark Gray)
- **Background:** #FFFFFF (Clean White)

### Typography
- **Headers:** Roboto Bold (24pt, 18pt, 14pt)
- **Body Text:** Open Sans Regular (12pt)
- **Code/Technical:** Fira Code (10pt)
- **Captions:** Open Sans Light (9pt)

### Layout Principles
- **Professional minimalist design**
- **Clear information hierarchy**
- **Generous white space**
- **Consistent alignment grid**
- **High contrast for readability**

### Visual Elements
- **Icons:** Lucide React icon set (consistent with application)
- **Borders:** Subtle 1px gray borders for sections
- **Shadows:** Soft drop shadows for depth
- **Progress bars:** For completion status
- **Code snippets:** Syntax highlighted examples
